🔮 Magic Circle Hands AR
An advanced browser-based AR hand tracking project built with HTML, CSS, JavaScript, and MediaPipe Hands. It uses your webcam to detect hands in real time and creates magical glowing circles, finger connections, and dual-hand energy effects.

✨ Features
Real-time hand tracking

Rotating magic circles on palms

Finger connection effects

Dual-hand energy bridge

Neon futuristic UI

Browser based interactive AR

🛠️ Tech Stack
HTML5

CSS3

JavaScript

MediaPipe Hands

Canvas API
